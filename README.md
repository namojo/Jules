# 블록 격파 게임 (Brick Breaker)

브라우저에서 실행되는 클래식 블록 격파 게임입니다.

## 게임 방법

- **방향키 (← →)** 또는 **마우스**로 패들을 움직여 공을 튕기세요
- 모든 블록을 파괴하면 레벨이 올라갑니다
- 공을 놓치면 목숨이 줄어듭니다
- 목숨이 0이 되면 게임 오버!

## 조작법

| 조작 | 설명 |
|------|------|
| ← / → | 패들 이동 |
| 마우스 | 패들 이동 |
| 스페이스바 | 게임 시작/일시정지 |

## 점수 시스템

- 상단 블록: 50점
- 중간 블록: 30점
- 하단 블록: 10점
- 레벨이 올라갈수록 공 속도 증가
