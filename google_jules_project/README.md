# Google Jules 프로젝트 (골격)

이 폴더는 `Google Jules`를 사용해 프로젝트를 만들기 위한 최소 골격입니다.

- 현재 파일들은 자리표시자(placeholder)입니다. `Google Jules`가 구체적으로 어떤 도구/템플릿을 말하는지 알려주시면, 그에 맞춰 실제 설정(의존성, 빌드 스크립트, 템플릿 등)을 추가하겠습니다.

다음으로 알려주세요:
- `Google Jules`가 정확히 무엇을 의미하나요? (예: Google에서 제공하는 특정 템플릿/툴, 내부 도구 등)
- 사용할 언어/런타임/패키지 매니저는 무엇인가요? (예: Python + venv, Node.js + npm)

현재 포함된 파일:
- `main.py`: 실행 가능한 간단한 진입점(자리표시자)
